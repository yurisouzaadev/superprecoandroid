## 📖 Sobre o projeto

O SuperCompras é um aplicativo Android desenvolvido com Kotlin e Jetpack Compose para auxiliar na organização de listas de compras.

O projeto foi desenvolvido com foco em praticar conceitos fundamentais do desenvolvimento Android moderno, utilizando arquitetura MVVM, gerenciamento de estado com StateFlow e construção de interfaces declarativas com Jetpack Compose.

Durante o desenvolvimento foram realizados 31 commits, registrando toda a evolução do projeto.

---

## ✨ Funcionalidades

✅ Adicionar novos itens à lista

✏️ Editar itens existentes

🗑️ Remover itens

✔️ Marcar itens como comprados

📋 Separação automática entre:

Itens Pendentes

Itens Comprados

🧠 Atualização reativa da interface utilizando StateFlow

🎨 Interface construída inteiramente com Jetpack Compose

---

## 🏗 Arquitetura

O projeto segue o padrão arquitetural MVVM (Model-View-ViewModel).

UI (Jetpack Compose)
        │
        ▼
   ViewModel
  (StateFlow)
        │
        ▼
  Lista de Itens


Essa separação permite maior organização do código e facilita futuras evoluções do projeto.

## 🚀 Como executar

Clone o projeto:

```bash
git clone https://github.com/yurisouzaadev/superprecoandroid.git
```

---


Abra o projeto no Android Studio.

Execute a sincronização do Gradle.

Inicie um emulador Android ou conecte um dispositivo físico.

Clique em Run ▶.
